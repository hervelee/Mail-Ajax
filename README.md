## Ce travail est à consulter en Localhost en suivant ce process :  
*1* `cloner ce repo`  
*2* `bundle install`  
*3* `rails db:create`  
*4* `rails db:migrate`  
*5* `rails db:seed`  
*6* `rails server`  
*7* `lancer localhost dans votre navigateur`
